# Changelog

All notable changes to the **AMSpiriT BASIC** extension are documented here.
This file is maintained by [Changesets](https://github.com/changesets/changesets).

## 0.2.0

- Pull BASIC from the emulator into a new editor (inverse of inject).
- Inject / Inject & Run / Reset & Run / Reset & Inject commands.
- Launch the emulator from VS Code with a status-bar connection indicator.
- Amstrad CPC BASIC syntax highlighting and line-number diagnostics.
- Bundled distribution: the extension ships as a single self-contained file.
